# Software selection

Currently, MixyLabs offers two software solutions dedicated for Mixy:

 - meej
 - Mixy Connect


## meej

meej is a background daemon with YAML config.  
Works on Windows and Linux.

Suitable for more technical people.

Download it from its [releases page](https://github.com/MixyLabs/meej/releases).  
Then, continue with its [setup guide](software_config/setup_meej.md).
